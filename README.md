# RetirePlan

A free, private retirement calculator and planner that runs entirely in your browser — no server, no data collection.

**[Run the calculator → bjeremy23.github.io/retireplan](https://bjeremy23.github.io/retireplan/)**

## Features

- Project account balances over time across multiple scenarios
- Model Social Security income
- Stress test market crashes
- Compare withdrawal strategies across below, average, and above market conditions

## Running locally

No build step required — just open `index.html` in a browser.

```bash
open index.html
```
